Workflow: describe_source a source — learn its real dimensions, measures, and joins — before you query it. Validate with query(execute:false) and iterate until clean, then run with a plain-English question describing what that query answers. A successful run returns a shareable link; present results by restating the question, then the rows, then the link. Do ranking, top-N, and member selection in Malloy, not in client code. Full detail: yo_help("explore/how-to").
